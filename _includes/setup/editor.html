(Recommended) Text Editor

A color-coded text editor makes it imple and easy to code without formating issues. Installation of one of the suggested text editors is recommended but not necessary.

  • Windows
  • MacOS
  • Linux

You can use Notepad++, Sublime Text or Visual Studio Code to edit the code.

You can use BBedit, Sublime Text or Visual Studio Code to edit the code.

You can use one of the following text editors in linux.

  • Gedit 
    sudo apt-get install gedit
  • Kate 
    sudo apt-get install kate
  • Sublime Text 
    sudo apt-get install sublime-text
You can install using command line from respective commands.
